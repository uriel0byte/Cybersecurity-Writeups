# BreachLab: Specter I Track
**OSINT & Passive Intelligence**

## Overview
Specter I is the Open Source Intelligence (OSINT) track on BreachLab, moving beyond basic search queries into professional-grade passive intelligence. This track focuses on source independence, operational security (OPSEC) against counter-investigators, and calibrated confidence—skills directly applicable to advanced threat intelligence and high-demand SOC roles. 

The track emphasizes defensible documentation, culminating in a written Intelligence report based on Berkeley Protocol standards. This directory contains the investigation reports and methodologies used to solve each level.
